Pothole Detection using Mobile Sensors

I'm currently working on my master's project - Pothole Detection using Mobile Sensors.

The aim of the project is to automate the process of reporting the potholes on the road. To achieve this, we plan to use ubiquitous smartphone devices widely used for navigation purpose. This information about potholes would be used to improve the experience of travel by integrating this information with the navigation algorithms.

We are planning to provide a web portal where users can view which area has more potholes and where road repairing is required most. This information would be helpful for municipal officers to take immediate action about potholes.

Food Locator

Integrated Disease Surveillance Program

Mobile Based ITS

Mobile Based Intelligent Transport System is an Android application that provides users route from destination A to destination B according to users priority (time, distance or comfort). The application is backed up by real-time traffic data, signal delays, road conditions etc.

To take into account multiple real-time factors, the system uses multivariate linear regression technique to calculate the best route.
